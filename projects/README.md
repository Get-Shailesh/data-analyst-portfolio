# 📊 Data Analyst Portfolio — Aryan Kumar

A collection of 6 end-to-end data analysis projects covering EDA, machine learning, time-series analysis, business intelligence, and marketing analytics.

---

## 🗂 Projects

| # | Project | Domain | Tools | Dataset |
|---|---------|--------|-------|---------|
| 1 | [E-Commerce Sales Trend Analysis](./project1_sales_trend/) | Retail | Python, Pandas, Matplotlib | Synthetic (realistic) |
| 2 | [Telecom Customer Churn Prediction](./project2_churn_model/) | ML / SaaS | Scikit-learn, Pandas | IBM Telco (Kaggle) |
| 3 | [Global COVID-19 Tracker](./project3_covid_tracker/) | Public Health | Pandas, Matplotlib | OWID (real, auto-downloaded) |
| 4 | [IBM HR Attrition Analysis](./project4_hr_attrition/) | HR Analytics | Pandas, Seaborn | IBM HR (Kaggle) |
| 5 | [FAANG Stock Volatility Analysis](./project5_stock_analysis/) | Finance | yfinance, Pandas | Yahoo Finance (live) |
| 6 | [Marketing Funnel & ROI Analysis](./project6_marketing_funnel/) | Marketing | Pandas, Matplotlib | Synthetic (realistic) |

## 🚀 Quick Start

### Clone this repo
```bash
git clone https://github.com/YOUR_USERNAME/data-analyst-portfolio.git
cd data-analyst-portfolio
```

### Run any project
```bash
cd project1_sales_trend
pip install -r requirements.txt
python sales_trend_analysis.py
```

Every project works standalone — just `cd` into it, install requirements, and run.

---

## 🛠 Common Requirements
All projects use Python 3.8+. Individual `requirements.txt` files are in each folder.

```bash
# Install all at once
pip install pandas numpy matplotlib seaborn scikit-learn yfinance requests
```
